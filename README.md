# Supermarket Billing Software

A desktop-based Supermarket Billing System developed using Java, JSwing, MySQL, and JDBC.  
This application helps generate bills, manage products, handle customers, and maintain sales records efficiently.

## Features
- Add, Update, Delete Products
- Product Category Management
- Customer Billing & Receipt Generation
- Auto Bill Calculation (Total, Tax, Discount)
- Store Daily/Monthly Sales Report
- MySQL Database Integration
- User-Friendly JSwing GUI

## Technologies Used
- Java
- JSwing (GUI)
- MySQL
- JDBC (Database Connectivity)

## How to Run This Project
1. Clone or download this repository.
2. Open the project in any Java IDE (NetBeans, IntelliJ, Eclipse).
3. Import the MySQL database file (**supermarket.sql**) into phpMyAdmin or MySQL Workbench.
4. Update your MySQL username and password inside the DB connection class.
5. Build and run the project.
